# WooCommerce Discord Redeem Bot

A Discord bot that allows users to redeem WooCommerce invoice IDs and automatically receive roles in a Discord server.

## Features
- Redeem command: `/redeem {order-id}`
- Verifies WooCommerce order via REST API
- Assigns role to Discord user
- Logs redemption to WordPress via REST webhook

## Deployment
- Host on [Render](https://render.com)
- Node.js v18+
